# 设计模式 #

## MVC ##

mvc设计模式指的是：

+ Model 模型

+ View 视图

+ Controller 控制器



>                 User (Web browser)
>                  ^         ^
>                  |          \
>                  |           \
>     Model <---> Controller---> View


MVC能够降低各模块之间的耦合性，极大程度上提高了代码的重用


## MTV ##

Django的设计模式 (各组件松耦合)


+ Model    模型 业务对象和数据库
+ Template 模板 页面展示
+ View     视图 业务逻辑
+ URL分发器


>                            / URL <---
>                           /         \
>                          /           \
>                      <--/             \
>      DB<---> views.py               /---> User
>                      \             /
>                       \---->Template
>
